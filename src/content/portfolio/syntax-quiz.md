- A TypeScript and JavaScript quiz app with five difficulty levels, from basic syntax fundamentals up to hard advanced concepts
- The entire codebase is generated by LLMs, supported by a robust pipeline of linting, static analysis, and unit and end-to-end tests to keep the output correct and maintainable
- Used LLMs to explore WebGL and shaders for the first time, producing animated visual effects I wouldn't have been able to write from scratch
- Provides [API documentation in Scalar using OpenAPI](http://syntax-quiz.netlify.app/api), backed by a type-safe oRPC + Hono API
  - Game state lives entirely in HMAC-signed URL parameters — no database needed, and progress can be saved or shared via a URL
- Build-time syntax highlighting via Shiki with a custom Vite plugin that pre-tokenises all code snippets and enforces WCAG AA contrast ratios, so highlighting is instantaneous with zero runtime cost
- Accessibility in vibe coding? It's possible! The quiz uses good HTML, contrasting colour schemes, ARIA live regions, keyboard navigation, haptic feedback patterns, 44px minimum touch targets, and full `prefers-reduced-motion` support
